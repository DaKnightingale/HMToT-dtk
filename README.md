Harvest Moon: Tree of Tranquility
===============================
A work-in-progress decompilation of Harvest Moon: Tree of Tranquility for Wii.

This repository does **not** contain any game assets or assembly whatsoever. An existing copy of the game is required.

Why Harvest Moon: Tree of Tranquility?
-------------
Harvest Moon: ToT is a game that is super near and dear to my heart. It is one of my favorite HM games in the entire series, and I feel like it doesn't get as much love as it deserves. This project aims to port HM:ToT to Windows (and potentially other platforms as well!) so that other people can enjoy this game ü§ç
